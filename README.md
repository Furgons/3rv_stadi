# 3RV Stādi

Šis ir skolas projekts — 3RV Stādi mājaslapa tomātu stādu katalogam.

## Projekta mērķis

Mājaslapas mērķis ir attēlot tomātu stādu katalogu, kur lietotājs var apskatīt šķirnes un filtrēt tās pēc vietas, augstuma, krāsas, augļa izmēra, ražošanas laika un pieejamības.

## Izmantotās tehnoloģijas

- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja

## Galvenās funkcijas

- Tomātu katalogs
- Filtri pēc kategorijām
- Atsevišķa tomāta apraksta lapa
- Kontaktforma
- Admin panelis
- Tomātu pievienošana, labošana un dzēšana
- Klientu jautājumu apskate admin panelī

## Palaišana

```bash
pip install flask
python app.py
```

Pēc palaišanas mājaslapa ir pieejama pārlūkā:

```text
http://127.0.0.1:5000
```
